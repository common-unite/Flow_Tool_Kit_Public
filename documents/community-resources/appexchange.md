# AppExchange

> Install Flow Tool Kit, check version history, and leave a review.

## Install Flow Tool Kit

Flow Tool Kit is available on the Salesforce AppExchange:

**[Flow Tool Kit (Form and Table Builder)](https://appexchange.salesforce.com/appxListingDetail?listingId=a0N4V00000HC4zCUAT)**

1. Click **Get It Now**.
2. Select your org (production or sandbox).
3. Choose **Install for Admins Only** (recommended).
4. Follow the prompts to complete installation.

See [Installation Guide](../getting-started/installation.md) for post-install setup steps.

## Leave a Review

If Flow Tool Kit is helping your team, we'd appreciate a review on AppExchange:

1. Go to the [Flow Tool Kit listing](https://appexchange.salesforce.com/appxListingDetail?listingId=a0N4V00000HC4zCUAT).
2. Scroll to the **Reviews** section.
3. Click **Write a Review**.
4. Rate the app and share your experience.

Reviews help other admins discover Flow Tool Kit and help us understand what's working well.
