# What's New

> The latest features and improvements in Flow Tool Kit.

## Latest Release: v3.211

Key highlights from the most recent release:

- **Formula Field Recalculation** — Form Builder now recalculates formula fields in real-time as users fill out forms, keeping dependent fields up to date without a page refresh
- **Enhanced Conditional Logic** — Improved performance and reliability for complex conditional visibility rules across sections and fields
- **Bug Fixes** — Multiple stability improvements across Form Builder, Flow Form, and Data Table components
